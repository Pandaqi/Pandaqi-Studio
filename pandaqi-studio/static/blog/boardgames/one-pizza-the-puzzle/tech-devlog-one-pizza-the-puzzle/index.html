Contains the technical devlog for “One Pizza The Puzzle”.

This game has randomly generated boards (press a button, get a board, print it, play!). Which is awesome, but also a lot of work to get right. The boards need to look good, be balanced, include any possible expansions/player counts/difficulty settings.

As such, this devlog isn’t done at the moment. If you really want to know how I made this, send me an email to say “hey, finish that thing now!” ;)
